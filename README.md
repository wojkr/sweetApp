# sweetApp
by [wojKr](https://github.com/wojkr)

Simple website built using Node.js Express, MongoDB and Passport auth.
This app requires MongoDB.

[Contact Me](mailto:test.wojkr@gmail.com?subject=[GitHub]%20React%20Crud%20Website)

## Overview
### Description
My first full webpage try. Deployed, with full DB and auth. 
This minimalist dessert rating website allows registered users to add desserts (with localtion), leave reviews, browse other users. 

Currently the project contains: 
* Interactive maps,
* Full auth,
* User accounts,
* Desserts (CRUD) with rating, locations and rated reviews,
* Uploading pictures to the Cloud 

### Technologies
* Node.js (Express), 
* MongoDB (mongoose), 
* EJS,
* MapBoxAPI (with GeoJSON), 
* Cloudinary,
* Bootstrap.
* Node packages:
  * Mongoose (MongoDB object modeling tool), 
  * Helmet (security), 
  * Sanitize-HTML (security), 
  * Connect-flash (Flash messages),
  * Joi (validation),
  * Passport (authentication)
